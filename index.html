Sistema de Recomendação - Backend API

Desafio Conty - Sistema Inteligente de Matching

Sistema de recomendação de criadores para campanhas publicitárias com algoritmo de scoring determinístico. Utiliza Jaccard similarity, sobreposição demográfica e normalização sigmoid para performance optimizada.

Arquitetura & Algoritmo:
  • API Layer: FastAPI + Pydantic para validação
  • Business Logic: Sistema de scoring determinístico
  • Data Layer: SQLAlchemy + SQLite para persistência
  • Jaccard Similarity: Matching de tags inteligente
  • Normalização Sigmoid: Performance balanceada
  • Sistema de Pesos: Tags(40%), Audiência(25%), Performance(20%)
Features Principais:
  • Endpoints RESTful: /recommendations, /api/v1/recommendations
  • Documentação Swagger: /docs e /redoc automáticos
  • Sistema de Justificativas: Explicação clara dos scores
  • Testes Automatizados: 6 testes cobrindo endpoints principais
  • Seeds Inteligentes: Dados fictícios realistas
  • Transparência Total: Score decomponível por fatores
Python 3.12+ FastAPI SQLAlchemy Pydantic SQLite Pytest
